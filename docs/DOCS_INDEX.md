## 🔍 搜索技巧

### 在文档中搜索

使用浏览器的查找功能（Ctrl+F / Cmd+F）搜索关键词：

- **端点名称**: 如 "POST /auth/login"
- **组件名**: 如 "ArticleCard"
- **错误信息**: 如 "ECONNREFUSED"
- **配置项**: 如 "DATABASE_URL"

### 常见主题

- **认证**: 搜索 "JWT", "token", "login"
- **数据库**: 搜索 "migration", "Alembic", "PostgreSQL"
- **部署**: 搜索 "Docker", "nginx", "SSL"
- **测试**: 搜索 "pytest", "test", "coverage"
- **性能**: 搜索 "optimization", "cache", "slow"
